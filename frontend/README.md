# FaceKey Frontend

React-Vite frontend for the FaceKey Building Access Control System.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

3. Build for production:

```bash
npm run build
```

## Features

- Dashboard with real-time statistics
- Building management
- Door access control
- User management with employee verification
- Access logs with filtering
- Real-time updates every 30 seconds

## Tech Stack

- React 18
- Vite
- React Router
- Bootstrap 5
- Font Awesome icons
